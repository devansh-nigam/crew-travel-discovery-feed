# AI Help / Reference Notes

Notes and references gathered during development to help future AI-assisted work on this project.

## Theme colors (extracted from reference app screenshots)

Source: color palette pulled from screenshots of a reference app's dark UI (chat/travel screens). Use this as the starting point when defining `src/constants/theme.ts` colors for this app's dark theme.

### Core backgrounds (dark mode foundation)
A deep, near-black tint with a warm brown/mauve undertone instead of pure neutral black.

| Role | Hex |
| --- | --- |
| Pure app background | `#0B0808` |
| Elevated cards/containers | `#272024` |
| Input fields & secondary buttons | `#34272B` |

### Accent & highlight colors
Signature dusty rose/mauve for primary actions, balanced by a couple of functional accents.

| Role | Hex |
| --- | --- |
| Primary call-to-action (e.g. send button) | `#904D4E` |
| Selected state (e.g. tag/pill) | `#2D2338` |
| System accent blue (native iOS elements) | `#0084FF` |
